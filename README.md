# Tars - a dead simple URL shortener

> A dead simple, no-frills, hobbyist, no guarantees URL shortener

[Tars.run] is nothing special. It's for learning and 
personal purposes only. That being said, its open-source and free to use. 
If you want to fork it, use it or contribute, you are welcome to do so.

This is the frontend for [Tars.run] and is built on top of the excellent 
[Next.js](https://nextjs.org) project. The backend is written in Go and can 
be found in this [repo](https://github.com/danielmichaels/shortlinks-go-frontend).

## How to use

First you'll need to install the packages
```bash
npm -i
# or
yarn 
```

To start the development server on the default port of 1988

```bash
npm run dev
```

Navigate to [localhost:1988](http://localhost:1988) once its built and get 
started.


[tars.run]: https://tars.run
